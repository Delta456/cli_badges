# CLI Badges

Generate Badges for your CLI.

## Features
- Make Beautiful Badges in CLI
- Works with all terminals
- Link support (soon)
- Variety of colors to choose from
- Written in TS with Deno
- many more coming...

## Installation

```ts
import { badges } from "https://deno.land/x/cli_badges@v0.0.1/index.ts";
```
